Helpful HTML Tags: A Cheat Sheet

Page Setup and Section Elements

Page setup elements establish the basic structure of your webpage. See the table below for helpful tags.

Opening Tag Closing Tag Notes
<!DOCTYPE html> N/A Establishes the document type as HTML. Must be the first line of the document. This element does not require a closing tag.
<head> </head> The head element contains metadata that is not displayed on the page, like the page title and link to CSS stylesheets. It is located below the <!DOCTYPE html> declaration and above the body of the webpage.
<body> </body> The body element contains items displayed by the web browser. There is only one body per html document.
<header> </header> The header element can hold things like the page title, author, navigation bar. It is used inside the body element.
<main> </main> The main element contains the main content of the page to be displayed and is used within the body element below the header section.
<footer> </footer> The footer element can contain copyright information, contact information, or other information less important to the overall purpose of the page and is used within the body element below the main section.

There are other elements that can help organize your webpages as well, like article, section, aside, and div, which will be covered by a future cheatsheet.

Lists

List elements organize information into lists for display by a web browser. There are two types of lists: ordered and unordered.

Ordered lists will be presented as numbered lists, while unordered lists will be presented as bulleted lists.

See the table below for helpful tags.

Opening Tag Closing Tag Notes
<ol> </ol> Defines the content between the opening and closing tags as an ordered (numbered) list.
<ul> </ul> Defines the content between the opening and closing tags as an unordered (bulleted) list.
<li> </li> These tags surround individual list items.
